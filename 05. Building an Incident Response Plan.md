## Detection & Analysis 

### NIST 800-61
Four categories for security event indicators 
  1.	**Alerts** - IDS, IPS
  2.	**Logs** - From server/workstation, SIEM, Syslogs
  3.	**Publicly Available Information** - New vulns in the wild
  4.	**People** - report of unusual activity
